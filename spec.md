MVP version (minimum project)
Build a personality assessment platform similar to Facet5.

Use the following technology stack.

Frontend

* Next.js
* TypeScript
* Tailwind CSS

Backend none
Data base none

Requirements

The platform should support:

* personality questionnaire
* sub-facet scoring
* response recording
* personality result

The frontend should include:

* multi-step questionnaire UI
* progress indicator
* result page showing personality profile
* radar charts displaying facet scores
* Description explain personality person follow on facet scores

Design a clean project structure for frontend.

code of conduct
* solid
* kiss

style of UI 
* Modern + clean + blue

Production version
Build a scalable personality assessment platform similar to Facet5.

Use the following technology stack.

Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

Backend

* NestJS
* Node.js

Database

* PostgreSQL
* Prisma ORM

Requirements

The system should support:

* personality questionnaire
* recording user responses
* scoring facets and sub-facets
* generating personality results
* REST API for assessments

Frontend features:

* questionnaire flow with multiple questions
* progress tracking
* results dashboard
* charts displaying personality scores

Backend features:

* assessment service
* scoring service
* result retrieval API

Database models should include:

* users
* assessment sessions
* questions
* answers
* facets
* subfacets
* scores

Generate a clean and scalable folder structure for frontend, backend, and database schema.
